# Custom PPT Templates

Put your PPT templates here.

Rules:
- Only `.pptx` files in this folder will be listed in the app.
- Preview images are managed in `../custom_template_previews`.
- PPT and preview must use the same file name stem.

Example:
- `custom_templates/investor_pitch.pptx`
- `custom_template_previews/investor_pitch.png`

Optional background image (used in export stage):
- `custom_templates/<same_stem>.png|jpg|jpeg`
- or `custom_templates/bgs/<same_stem>.png|jpg|jpeg`
