# Custom Template Previews

Put your preview images here.

Mapping rule:
- Preview file name must match the PPTX file name (same stem).
- Example:
  - `custom_templates/investor_pitch.pptx`
  - `custom_template_previews/investor_pitch.png`

Supported image types:
- `.png` `.jpg` `.jpeg` `.webp`

Only templates in `custom_templates` are listed in the app.
